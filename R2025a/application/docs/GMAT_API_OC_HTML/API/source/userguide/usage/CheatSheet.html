Object Creation¶

Python

MATLAB

obj = gmat.Construct(<type>,<name>)

obj = GMATAPI.Construct(<type>, <name>)

cmd = gmat.Command(<type>,<desc>)

cmd = GMATAPI.Command(<type>, <desc>)

Examples

counter = GMATAPI.Construct (“Variable”,”Counter”)

maneuver = gmat.Command (“Maneuver”, “Burn(Sat)”)

assign = GMATAPI.Command (“GMAT”, “Counter = 0”)
